# TTcut v1.0.1

本版本增加了手动下载和导入组件功能，方便网络受限时完成本地组件配置。

## 更新内容

- 增加手动下载入口，可前往组件下载页面并直接下载 FFmpeg 组件。
- 增加多文件导入组件功能，支持模型、CPU/CUDA 分析运行时和 FFmpeg 组件。
- 导入组件后自动完成完整性校验、解压、组件自检和本地配置。
- 重新检测组件时，会明确提示缺少的组件。
- 下载分析组件时提示：打开虚拟网卡或 TUN 模式加快下载速度。

## 安装

下载 `TTcut-1.0.1-x64-Setup.exe`，安装前请核对 Release 中的 `SHA256SUMS.txt`。
